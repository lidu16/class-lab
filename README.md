# Quiz Game

A fun and interactive quiz game built with HTML, CSS, and JavaScript. Test your knowledge with multiple-choice questions across various topics!

## Features

- **Multi-page Navigation**: Clean separation between start screen, quiz questions, and results
- **Progress Tracking**: Visual progress bar showing quiz completion
- **Instant Feedback**: Answers are highlighted immediately after selection
- **Score Calculation**: Automatic scoring with encouraging messages
- **Responsive Design**: Works on different screen sizes
- **Smooth Animations**: CSS transitions for better user experience

## How to Play

1. Open `index.html` in your web browser
2. Click "Start Quiz" to begin
3. Answer each question by clicking on your choice
4. See immediate feedback (green for correct, red for incorrect)
5. View your final score and message on the results page
6. Click "Restart Quiz" to play again

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and animations
- **JavaScript**: Game logic and interactivity
